# AI_QA3 – Rule-Based Library Chatbot

## Course: Artificial Intelligence  
## Unit: 3  
## Assessment: Qualitative Assessment – 3  

This project implements a **rule-based Library Chatbot**
using predefined rules and keyword matching,
similar to early AI systems.

### Features
- Responds to library queries
- Rule-based reasoning
- No AI / ML / NLP libraries used

### How to Run
1. Compile: `javac LibraryChatbot.java`
2. Run: `java LibraryChatbot`
